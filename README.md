# Tahial Climbing

Sitio web estático de Tahial Climbing con catálogo, galería, videos promocionales y carrito local.

## Publicación con GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube este proyecto a la rama `main`.
3. En GitHub, entra a **Settings > Pages**.
4. En **Build and deployment**, selecciona:
   - **Source:** Deploy from a branch
   - **Branch:** `main` y carpeta `/ (root)`
5. Guarda los cambios.
6. GitHub publicará el sitio en una URL como:
   `https://TU-USUARIO.github.io/TU-REPO/`

## Dominio propio

Cuando tengas tu dominio:

1. Crea un archivo llamado `CNAME` en la raíz del proyecto con tu dominio, por ejemplo:
   `www.tudominio.com`
2. En tu proveedor de dominio, apunta el dominio a GitHub Pages.
3. En GitHub Pages, activa HTTPS cuando esté disponible.
